# UnixCore-6502

Just for Fun

Questo progetto nasce dalla passione che ho sempre avuto dalla più tenera età per il mondo della tecnologia, che poi, a partire dai tredici anni,
si è declinata più nello specifico in amore per l'informatica.
Questa iniziativa rappresenta una sfida che mi sono posto con me stesso per riuscire a realizzare una macchina che potesse somigliare ad un sistema Unix
ad un livello basilare. Più nello specifico, il file cardine di questo progetto è quello che contiene il codice per Arduino, senza il quale non
sarebbe stato possibile implementare un sistema operativo. Ci si aspetterebbe, in un progetto del genere, l'utilizzo del set completo di IC per realizzare
l'architettura della macchina, ma in questo caso corre in soccorso Arduino.
